This page describes the open source projects on Matthew Rocklin's github page.

If you'd prefer you can just go directly to the standad github page.

  • Projects started by me:
    • Matrix Algebra in Maude: A tiny DSL for simplifying matrix expressions. It is written in the Maude system using rewrite rules. The eventual goal is to wrap the complexity of using BLAS and LAPACK into an easily expressible mathematical language.
    • APE: Static scheduling of array operations onto heterogeneous hardware. Wraps an MPI layer around the Theano project. Uses tompkins (see below) for scheduling.
    • tompkins: A scheduler for computations on heterogeneous architectures including dependencies between tasks (represented as a DAG), and communication times. It solves the problem using an integer linear programming method described in an article by Tompkins. This code attempts to translate his math into clear Python code. Section and equation numbers are included. Uses the pulp library to describe the ILP.
    • Shallow Water: A time dependent solver for the Shallow Water equations. This code uses numpy or Theano for low level computation. It was designed to demonstrate how mathematical solvers can be written clearly in a high-level language and use libraries for efficient low-level computation. mailFeed: A simple Python library to automate the collection of e-mail from a remote server.
    • alettertofaye: A letter attempting to explain common sense internet security to a non computer-scientist.
  • Projects to which I contribute
    • SymPy: A symbolic mathematics library within Python. Similar to Mathematica or Maple. I contribute to the statistics and linear algebra modules.
    • Theano: A library that bridges the symbolics-numerics gap. They self-style themselves as a "Math Compiler." My contributions to this project so far are trivial. I list it here because it is important to the APE project.
